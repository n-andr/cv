# CV (EN/DE) — GitHub Pages

This repository hosts a bilingual CV website (English + German) deployed via GitHub Pages as a **project site**:

- https://n-andr.github.io/cv/

## Structure

- `/en/` English CV
- `/de/` German CV
- `/assets/` shared CSS + renderer + CV data

## Local preview

Open `en/index.html` or `de/index.html` in a browser.

## Focus switch (optional)

You can add a query param to emphasize a focus:

- `?focus=fullstack`
- `?focus=backend`
- `?focus=frontend`

Example:
- https://n-andr.github.io/cv/en/?focus=fullstack

## Deploy (GitHub Pages)

Repo settings → Pages:
- Source: **Deploy from a branch**
- Branch: `main`
- Folder: `/ (root)`
